# Scripts

## `csv_to_db.py`
- **input:** `databases/job_ads.csv`
- **output:** `databases/job_ads.db`
Создает по `csv` образу датасета `SQLite3` версию.

## `csv_make_mini.py`
- **input:** `databases/job_ads.csv`
- **output:** `databases/mini_jobs.csv`
Создает по `csv` образу датасета мини-версию (100 случайных строк) для тестирования.
