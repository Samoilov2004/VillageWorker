ML-микросервис для платформы цифрового хаба сельских территорий.

## Назначение

Сервис реализует интеллектуальную обработку данных и предоставляет API для следующих задач:

- рекомендации похожих вакансий, резюме и других сущностей;
- пре-модерация пользовательского контента;
- поисковая выдача и ранжирование объявлений;
- хранение и обновление внутреннего индекса сущностей, используемого для поиска и рекомендаций.

Сервис является отдельным компонентом архитектуры и вызывается backend'ом по HTTP API.

Все ML-методы публикуются под префиксом `/api/ml`. Примеры:
- `/api/ml/index/upsert`
- `/api/ml/search`
- `/api/ml/recommend/similar`
- `/api/ml/moderation/check`
Служебные методы доступны отдельно:
- `/health`
- `/ready`

## Запуск микросервиса
```bash
python3 -m venv .mlvenv
source .mlvenv/bin/activate
pip install -r ml_service/requirements.txt

uvicorn ml_service.app.main:app --reload --host 127.0.0.1 --port 8001
```

## Запуск тестов
```bash
pytest ml_service/tests -v
```

## Документация API
После запуска проекта будет доступна техническая документация к API

- Swagger UI: `http://127.0.0.1:8001/docs`
- ReDoc: `http://127.0.0.1:8001/redoc`
- Health-check: `http://127.0.0.1:8001/health`
